Roadmap

The FilmVault project is in active development. The focus in v1.0 is on the basic movie discovery experience. Features planned for future versions (v2.0 and later) include:

User Auth: A login/registration system for a personalized experience.

My Favorites/Watchlist: A "My List" page where users can save their favorite movies.

TV Shows: Lists popular series with TMDB API integration.

Advanced Search: A detailed search engine based on movies, series, and actors.

-------------------------------------------------------------------------------------------------------------------
Yol Haritası (Roadmap)

FilmVault projesi aktif geliştirme aşamasındadır. v1.0 sürümünde temel film keşif deneyimine odaklanılmıştır. Gelecek sürümler (v2.0 ve sonrası) için planlanan özellikler şunlardır:

Kullanıcı Üyeliği (Auth): Kişiselleştirilmiş deneyim için giriş/kayıt sistemi.

Favorilerim / İzleme Listesi: Kullanıcıların beğendikleri filmleri kaydedebileceği "Listem" sayfası.

Diziler (TV Shows): TMDB API entegrasyonu ile popüler dizilerin listelenmesi.

Gelişmiş Arama: Film, dizi ve oyuncu bazlı detaylı arama motoru.

Technologies Used
Backend: C# / ASP.NET Core 8.0

Frontend: Razor Pages, HTML5, Tailwind CSS

Data Source: The Movie Database (TMDB) API

JSON Processing: System.Text.Json

How to Run:
Clone or download the project.

Get a free API key from the TMDB website.

Open the Pages/Index.cshtml.cs file and paste your own key into the ApiKey field.

Open and run the project in Visual Studio (F5).
